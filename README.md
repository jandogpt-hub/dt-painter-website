# DT Painter Website

Professional painting services website for DT Painter in Coral Springs, FL.

## Tech Stack

- **Backend:** PHP 8.x
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Hosting:** Hostinger (via GitHub auto-deploy)

## Local Development

1. Install a local PHP server (XAMPP, MAMP, or similar)
2. Clone this repository to your `htdocs` folder
3. Access via `http://localhost/dt-painter-website/website/`

## Deployment

This repository is connected to Hostinger via Git integration. Pushing to `main` automatically deploys to the live site.

## Structure

```
website/
├── index.php              # Homepage
├── about.php              # About page
├── contact.php            # Contact page
├── services/              # Service pages
├── landing/               # Promo landing pages
├── gallery/               # Photo gallery
├── includes/              # PHP partials (header, footer, nav, config)
└── assets/                # Static files (CSS, JS, images)
```
